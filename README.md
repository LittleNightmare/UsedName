# UsedName

一个针对FFXIV无备注名情况开发的插件。目前可以记录曾用名，添加昵称 *开发中*

## How to use

使用'/pname update'更新好友列表

使用'/pname search xxxx'搜索xxxx的曾用名

使用'/pname nick xxxx aaaa'设置xxxx的昵称为aaaa，仅支持好友
